MOBILE BACKGROUND IMAGES
========================

Desktop:
  - hand pick 1.jpeg   (top)
  - hand pick 2.jpeg   (bottom)
  - image 3.jfif       (Impact & Contact)

Mobile (required filenames in this folder):
  - hand pick mobile 1.jpeg  → About / Vision / top sections
  - hand pick mobile 2.jpeg  → Services / Partner / Immersion
  - image 3.jfif             → Impact & Contact (same as desktop)

Recommended size for each mobile file:
  - Width:  1080 px (min 828 px)
  - Height: 1920 px (9:16 portrait)
  - Format: JPEG or JFIF
  - Quality: 75–85% (keep each file under ~400 KB if possible)

Tips:
  - Crop the same photo you use on desktop, centered on the subject.
  - Portrait (tall) works better than landscape on phones.
  - Do not use spaces in new filenames if possible; current names match the site.

After uploading, push to GitHub and redeploy on Vercel.
